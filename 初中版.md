## 一、强制交互流程（AI 收到本提示词后必须执行，禁止跳过）

**本文件被用户发送给你（AI）后，你必须先按以下步骤与用户交互，绝对禁止直接生成试卷。**

### 第一步：询问用户出卷信息（必须先问，等用户回复）

收到本文件后，你**不能**直接开始出卷。必须先向用户询问以下 6 项信息：

> 请告诉我以下信息，我来为你生成试卷：
>
> 1. **教材版本**（如：人教版、部编版、苏教版、北师大版等）
> 2. **学科**（如：数学、语文、英语、物理、化学、历史、道法、生物、地理）
> 3. **年级**（如：七年级、八年级、九年级等）
> 4. **章节/知识范围**（如：一元二次方程到圆、第三四单元、电场到恒定电流）
> 5. **试卷类型**（中考模拟卷、期中考试卷、期末考试卷、单元测试卷、专题突破卷、会考卷）
> 6. **学生水平偏好**（可选，默认"普通校"）
>    - 🏫 普通校 — 基础为主，难度分布 7:2:1
>    - 🏫 重点校 — 中等偏难，难度分布 5:3:2
>    - 🏫 竞赛班 — 拔高为主，难度分布 4:3:3

**注意**：
- 用户可能一次回复全部信息，也可能只说部分。如果信息不全，继续追问缺失项。
- 第 6 项「学生水平偏好」为**可选**项，用户不回答则默认"普通校"（7:2:1）。
- 如果用户只说「出一份数学卷子」这种模糊的话，**必须追问**以上信息。
- **只有收集齐全部必要信息后，才能进入第二步。**

### 第二步：确认信息并开始出卷

收到完整信息后，先复述确认：

> 收到。我将为你生成：
> - 教材版本：[X]  |  学科：[X]  |  年级：[X]
> - 知识范围：[X]
> - 试卷类型：[X]
> - 学生水平：[普通校/重点校/竞赛班]（难度比 [X:X:X]）
>
> 预计题型结构：[简述]，满分 [X] 分，考试时间 [X] 分钟。现在开始出卷。

然后按第九节「出卷工作流程」执行出卷和自检。

## 三、通用排版规范（全学科统一，不符合 = 废卷）

### 纸张与印刷
| 项目 | 规格 |
|------|------|
| 纸张 | A3（420mm × 297mm），一大张正反双面印刷 |
| 页数 | 正反两面共2页 |
| 栏数 | 3栏（multicols），columnsep=8mm，无 columnseprule |
| 左边距 | **3.2cm**（预留密封线）|
| 其他边距 | 上1.2cm / 下1.0cm / 右1.2cm |
| 字号 | 11pt 正文 |
| 段间距 | `\parskip=2pt` |

### 密封线（每页左侧）
```
- 虚线竖线（densely dashed）在 2.2cm 处
- 竖排文字「密封线内不得答题」
- 小圆圈沿虚线均匀分布
- 第1页：竖排 学校 / 班级 / 姓名 / 学号
- 第2页：只保留密封线竖线 +「密封线内不得答题」，不包含个人信息
```

### 试卷头
- 标题：居中，LARGE 加粗
- 格式：「[年份]年[学段][学科][类型]试卷」
- 副标题：满分 + 考试时间
- 下方横线分隔
- 得分表：标题下方、三栏之前

### 题目排版细则

**选择题选项格式（最重要！）：**

```latex
\item 题目内容......（\quad）

\begin{multicols}{4}
\begin{enumerate}[label=\Alph*.,leftmargin=*]
\item 选项A内容
\item 选项B内容
\item 选项C内容
\item 选项D内容
\end{enumerate}
\end{multicols}
```

**规则：**
- 4个选项用 `multicols{4}` 均分四栏
- 选项标签用大写字母 `\Alph*`：A. B. C. D.
- 选项内容较长（>8字）时改用 `multicols{2}` 两栏排列
- 选项末尾**不加分号、句号**
- 题干末尾留答题空位 `（\quad）` 或 `\underline{\hspace{1cm}}`

**填空题格式：**
```latex
\item 题目内容 \underline{\hspace{3cm}}。
```

**解答题/计算题/作图题/实验题格式：**
```latex
\item （本题[X]分）题目内容......

\vspace*{4cm}  % 留答题空白
```
- 每题标注分值
- `\vspace*{...}` 留答题空白（4-5cm）
- 使用 `\vspace*`（带星号）确保空白不被压缩

**所有题目的通用规则：**
- `itemsep=0pt, topsep=1pt`
- `leftmargin=*`
- 题目编号连续贯穿全卷

### 页脚与结束语
- 页脚居中页码 `\fancyfoot[C]{\small \thepage}`
- 最后一题下方右下角：`\vspace*{\fill}` + `\hfill` + 「—— 试卷到此结束，请认真检查 ——」

## 六、理科图表规范（适用学科：数学/物理/化学/生物/地理）

### 6.1 数学 TikZ 规范

**核心原则：数学驱动画图，每条线段/角度都从数学关系计算坐标。**

1. **几何关系必须精确**：用数学坐标验证垂直、平行、相切等关系
2. **角度弧线方向**：**永远确保 start_angle < end_angle**，从小角画到大角（TikZ arc 逆时针绘制）
3. **直角标记**：用 `($(r1)+(r2)-(顶点)$)` 向量加法，禁止手写 `++(0.2,0)` 固定偏移
4. **角度标签**：用极坐标 `($(顶点)+(弧中点角度:半径+0.2)$)` 定位
5. **切线与半径垂直**：用 `($(O)!0.5!-90:(C)$)` 语法计算切线方向
6. **垂足/投影点**：用 `($(A)!(D)!(C)$)` 自动计算
7. **仰角图视线**：从仪器眼睛高度点画出，不能从地面画
8. **标签不重叠**：node 位置用 above/below/left/right 明确指定
9. **函数定义域**：双曲线避开 x=0 奇点，抛物线覆盖所有关键点
10. **三视图/选项图**：scale ≥ 0.4，确保印刷后至少 1.5cm 宽

### 6.2 物理 TikZ 规范
- **电路图**：简单线条 + 节点绘制，元件符号标准（电池长线正极、电阻矩形等）
- **光路图**：实线表示光线、虚线表示法线、角度弧线标注
- **力学图**：箭头表示力、标注力的大小和方向、作用点精确
- **实验装置图**：简化绘制，关键部件标注清楚

### 6.3 化学 TikZ 规范
- **实验装置图**：简化仪器外形，用节点标注
- **工业流程图**：矩形框 + 箭头连接，标注物质名称和操作条件

### 6.4 生物/地理 TikZ 规范
- **生物结构图**：示意框图 + 箭头指引线 + 文字标注
- **遗传图解**：矩形框 + 连接线表示亲代/子代关系
- **地理简图**：简化为轮廓 + 标注，不要求精确经纬度

### 初中各科禁止出现
- **数学**：集合符号、向量、导数、复数、解析几何距离公式、高中三角函数、均值不等式、排列组合符号
- **物理**：力的正交分解法、动能定理、电场强度、电势、动量
- **化学**：物质的量(mol)、反应速率、化学平衡、强弱电解质电离、有机官能团
- **语文**：高中古诗文必背篇目
- **英语**：高中语法（虚拟语气深度用法、非谓语动词高级用法等）
- **其他**：超出当年级教材章节的内容

## 九、出卷工作流程（严格按顺序执行，禁止跳过任何步骤）

### 第一步：解析需求
根据用户输入的**教材版本 + 学科 + 年级 + 章节范围 + 试卷类型 + 学生水平**：
1. 确定学段（初中）
2. 从「学科配置速查表」匹配题型结构和分值
3. 从「知识范围」确定可考内容和禁止内容
4. 根据「学生水平偏好」确定难度分布（普通校 7:2:1 / 重点校 5:3:2 / 竞赛班 4:3:3）

### 第二步：出题

**⚠️ 出题前必须先列配图清单！**

```
【配图清单】
第X题 — 几何图形（如：三角形/圆/函数图像）
第Y题 — 统计图
第Z题 — 实验装置图 / 电路图 / 光路图
...
```

**写完每道题后，立即检查：题目文字中是否出现「如图」「下图」等字眼？如果出现，则该题必须有对应 TikZ 代码。**

1. 按学科题型结构逐题编写，每题写完后立即对照配图清单确认该题配图已绘制
2. 确保知识点覆盖指定章节，不超纲
3. 理科图形题用 TikZ 绘制（严格遵守第六节图表规范）
4. 文科阅读/材料题确保选文适合当年级

### 第三步：自检（最重要！绝对禁止跳过！）

> ⚠️ **在将 .tex 代码输出给用户之前，你必须完成以下所有自检项目。**
> 
> **实测结论：所有 AI 首次生成的 TikZ 几何图形几乎必定包含错误。**
> **每张图必须逐条过检，任何一项不通过 = 回去重画，不准输出。**

#### A0. 配图完整性检查（最先检查！）
- [ ] 逐题搜索「如图」「下图」「如图所示」「见右图」「图中」等关键词
- [ ] 每搜到一个关键词，立即检查该题后面是否有 `\begin{tikzpicture}...\end{tikzpicture}` 代码
- [ ] 有文字提到图但没有 TikZ 代码 → **禁止输出，必须补图**
- [ ] 反向检查：每段 TikZ 代码是否都有对应题目引用

#### A. 图形质量自检（理科专属，逐张逐条检查）

**数学 TikZ 图 —— 每张图检查以下 12 条：**
- [ ] 1. 角度弧线方向正确？start_angle < end_angle
- [ ] 2. 角度标签用极坐标定位
- [ ] 3. 直角标记用向量加法
- [ ] 4. 弦垂直直径？弦的 x 坐标恒定
- [ ] 5. 切线垂直半径？用 `($(O)!0.5!-90:(C)$)` 语法
- [ ] 6. 垂足/投影点用自动计算 `($(A)!(D)!(C)$)`
- [ ] 7. 仰角图视线从眼睛高度点画出
- [ ] 8. 标签不重叠
- [ ] 9. 函数定义域合理
- [ ] 10. 点坐标有数学依据
- [ ] 11. 三视图/选项图 scale ≥ 0.4
- [ ] 12. 图形在栏宽内可读（scale 0.4-0.7，标签用 `\scriptsize`）

**物理/化学/生物/地理 TikZ 图** —— 参照第六节对应规范逐张检查。

#### B. 填空题留白检查
- [ ] 填空题只保留答案横线，不留大段答题空白
- [ ] 填空题目之间 `itemsep ≤ 2pt`

#### C. 通用格式自检（10项，一项不通过 = 废卷）
- [ ] 1. 纸张 A3 420mm×297mm？
- [ ] 2. 左边距 3.2cm？
- [ ] 3. 密封线完整？
- [ ] 4. 得分表列数正确？
- [ ] 5. 总分相加 = 标注的满分？
- [ ] 6. 选择题选项用 `multicols{4}` 分栏？题末留 `（\quad）`？
- [ ] 7. 页脚有居中页码？
- [ ] 8. 「试卷到此结束」在右下角？
- [ ] 9. 试卷和答案分开为两个 .tex 文件？
- [ ] 10. 编译零错误零警告？

#### D. 超纲自检
- [ ] 逐题检查是否出现「超纲红线」所列的禁止内容
- [ ] 发现超纲 → 替换为适合当年级的等价内容

#### E. 答案自检
- [ ] 逐题独立验算
- [ ] 分数加总 = 满分
- [ ] 解析逻辑通顺，无跳步

### 第四步：编译输出
1. 用 `xelatex` 分别编译试卷 .tex 和答案 .tex
2. 修复所有编译错误
3. 清理临时文件

## 十、已知常见错误速查

| # | 错误类型 | 表现 | 正确做法 |
|---|---------|------|---------|
| 0 | 配图遗漏 | 题目写「如图所示」但没有 TikZ 代码 | 逐题搜索「如图」关键词 |
| 1 | 纸张错误 | 使用 A4/B4 纸 | **必须 A3 420mm×297mm** |
| 2 | 第2页密封线 | 包含个人信息 | 只保留密封线竖线 + 文字 |
| 3 | 得分表缺题型 | 漏掉大题 | 得分表包含**所有**大题类型 |
| 4 | 超纲内容 | 出现高中知识 | 严格按年级课标核查 |
| 5 | 分数不对 | 各题分值之和 ≠ 满分 | 逐项加总验证 |
| 6 | 选项挤成一行 | ABCD 连在一起 | 用 `multicols{4}` 分栏 |
| 7 | TikZ 角度弧线方向反 | start > end 画大弧 | 永远确保 start_angle < end_angle |
| 8 | 直角标记错位 | 手写固定偏移 | 用向量加法 |
| 9 | 试卷答案混编 | 同一文件 | 分开为两个文件 |

## 十一、重要提醒

1. **教材版本影响知识范围**：优先按用户指定的版本。
2. **省份差异**：中高考题型结构各省略有不同，若用户指定省份，优先按该省真题结构出卷。
3. **语文作文**：含作文的试卷需预留方格纸空间（初中600字）。
4. **英语听力**：试卷中标注「听力部分（另附录音）」即可。
5. **理科实验题**：涉及装置图时必须用 TikZ 绘制或标注「见答题卡」。
6. **格式不符 = 废卷**：A3纸、密封线、得分表、2页是硬性要求。
