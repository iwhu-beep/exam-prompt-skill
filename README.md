# 全学科试卷命题技能

## 介绍

这是一个面向 **AI 大语言模型** 的试卷命题提示词工程（Prompt Engineering）项目。

传统出卷流程中，教师需要手动选题、排版、绘图、核分，一份高质量模拟卷往往耗时数小时。本项目将小初高全学段全学科的命题规范、课标红线、排版标准整合为一份结构化提示词，发送给 AI 后即可自动生成符合学校考试标准的 LaTeX 试卷源码，编译后直接交付印刷。

**核心解决的问题：**

- **命题规范碎片化** — 课标知识范围、超纲红线、题型结构、难度分布分散在不同文档中，本提示词一次性整合
- **AI 排版不可控** — 通用 AI 输出的试卷常常 A4 纸、选项挤一行、缺密封线/得分表，本提示词通过强制排版规范解决格式废卷问题
- **理科配图错误率高** — TikZ 几何图形角度弧线方向反、直角标记错位是 AI 通病，本提示词包含 12 条逐项自检清单

**适用人群：** 中小学教师、教培机构教研员、学校教务处、家长辅导。

**效果：** 输入 5 项出卷参数（版本/学科/年级/章节/类型），AI 在 1-3 分钟内输出符合 A3 大纸双面印刷标准的 LaTeX 试卷 + 参考答案。

## 快速开始

### 方式一：Claude Code（终端内直接出卷，.tex 文件一键编译）

Claude Code 支持文件引用和 Skill 机制，无需复制粘贴，在终端内完成全流程：

**方法 A：直接引用文件（最简单）**

1. **下载文件到项目目录**
2. **在 Claude Code 中输入**：

```
@全学科试卷命题技能.md 帮我生成一份试卷
```

3. Claude Code 读取提示词后自动询问出卷信息，回复后生成 .tex 文件
4. **编译**：`xelatex 试卷.tex && xelatex 答案.tex`

**方法 B：注册为 Skill（长期使用）**

将文件路径写入 CLAUDE.md 或 memory，之后每次只需说「出一份数学期中卷」即可触发：

```markdown
# CLAUDE.md
- 试卷命题：参考 @/path/to/全学科试卷命题技能.md
```

> Claude Code 模式下，AI 会严格遵循提示词中的自检流程，尤其 TikZ 几何图形的数学坐标验证比其他 AI 更可靠。

### 方式二：豆包超能模式（推荐，无需编译，直接下载 PDF）

豆包的「超能模式」支持上传文件后直接生成可下载的 PDF：

1. **下载提示词文件** — 下载 [`全学科试卷命题技能.md`](./全学科试卷命题技能.md)
2. **上传到豆包** — 在豆包中开启「超能模式」，上传该 md 文件
3. **告诉它考点范围** — 例如：「人教版八年级数学，一元二次方程到圆，期中测试卷」
4. **直接下载 PDF** — 豆包自动生成排版好的 PDF 试卷，点击下载即可

> 超能模式省去了 LaTeX 编译环节，不需要安装任何软件，手机端也能操作。

### 方式三：通用 AI 对话（输出 LaTeX 源码，需自行编译）

适用于 ChatGPT / DeepSeek / Kimi 等所有 AI：

1. **复制提示词** — 打开 [`全学科试卷命题技能.md`](./全学科试卷命题技能.md)，复制全部内容
2. **粘贴到 AI 对话框** — 发送后 AI 会主动询问出卷信息
3. **回答 5 个问题** — 教材版本、学科、年级、章节范围、试卷类型
4. **获得 .tex 文件** — AI 生成 LaTeX 源码，编译后得到 A3 排版 PDF

## 覆盖范围

| 学段 | 年级 | 学科 |
|------|------|------|
| 小学 | 1-6 年级 | 数学、语文、英语、科学、道法 |
| 初中 | 7-9 年级 | 数学、物理、化学、语文、英语、历史、道法、生物、地理 |
| 高中 | 10-12 年级 | 数学、物理、化学、语文、英语、历史、政治、生物、地理 |

支持人教版、部编版、苏教版、北师大版等主流教材版本。

## 试卷类型

- 中考/高考模拟卷
- 期中/期末/单元测试卷
- 专题突破卷
- 会考/学考卷

## 排版特性

- A3 大纸双面印刷，三栏排版
- 密封线（虚线 + 学校/班级/姓名/学号）
- 得分表自适应学科题型
- 理科图形用 TikZ 精确绘制
- 严格不超纲，难度分布 6:2.5:1.5

## 编译 LaTeX（方式一和方式三需要）

方式二（豆包超能模式）直接下载 PDF，无需此步骤。

```bash
xelatex 试卷.tex    # 编译试卷
xelatex 答案.tex    # 编译参考答案
```

macOS 用户推荐安装 [MacTeX](https://tug.org/mactex/)，Windows 用户推荐 [TeX Live](https://tug.org/texlive/)。

## 注意事项

- **豆包超能模式最省事** — 上传 md 文件直接出 PDF，零门槛，适合不熟悉命令行的用户
- **Claude Code 质量最高** — 终端内操作，严格自检验证，TikZ 图形数学坐标计算更精确
- **其他 AI 输出 .tex 源码** — ChatGPT/DeepSeek/Kimi 等不能直接生成 PDF，需自行编译
- **首次使用建议先试单元测试卷** — 题量少、编译快，验证环境没问题再做完整模拟卷
- **如遇编译错误** — 将错误信息发回 AI，让它修复
- **不同 AI 效果有差异** — Claude/GPT-4 对 TikZ 图形和 LaTeX 格式把控更稳定
