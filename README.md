## 快速开始

### 方式一：豆包超能模式（推荐，无需编译，直接下载 PDF）

1. 下载对应学段的 `.md` 文件（如只需要出小学试卷，下载 `小学版.md` 即可）
2. 在豆包中开启「超能模式」，上传该 md 文件
3. 告诉它考点范围，如：「人教版四年级数学，第三四单元，期末测试卷」
4. 直接下载 PDF

### 方式二：Claude Code（终端内出卷 + 直接编译 PDF）

```
@小学版.md 帮我生成一份人教版四年级数学期末卷，范围是小数的意义和性质到三角形，普通校水平，编译成 PDF
```

### 方式三：通用 AI 对话（输出 LaTeX 源码，需自行编译）

1. 复制对应学段的 `.md` 文件全部内容
2. 粘贴到 AI 对话框
3. 回答 AI 询问的出卷信息（教材版本/学科/年级/章节/类型/学生水平）
4. 获得 .tex 文件，用 `xelatex` 编译

## 编译 LaTeX

```
xelatex 试卷.tex    # 编译试卷
xelatex 答案.tex    # 编译参考答案
```

macOS 推荐 MacTeX，Windows 推荐 TeX Live（需 2020-10-01 之后的内核版本以支持 `\AddToHook`）。
